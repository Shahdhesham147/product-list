# 🛒 Product List Project

A simple **e-commerce front-end project** built using HTML, CSS, and JavaScript.  
It allows users to browse products and manage a fully functional shopping cart.

---

## ✨ Features

- Display products dynamically
- Add items to cart
- Increase / decrease item quantity
- Remove items from cart
- Real-time total price calculation
- Order confirmation modal
- Reset cart after confirming order
- Responsive design for different screen sizes
- Hover effects for better UI experience

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
