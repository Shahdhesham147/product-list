# 🛒 Product List – Shopping Cart App

A fully functional **front-end e-commerce product list application** built with **HTML, CSS, and Vanilla JavaScript**.  
This project simulates a real shopping experience including cart management, dynamic updates, and order confirmation.

## ✨ Features

- Dynamic product rendering from local `data.json`
- Add products to cart
- Increase / decrease item quantity
- Remove individual items from cart
- Real-time cart total calculation
- Order confirmation modal
- Reset cart after successful order
- Responsive design for all screen sizes
- Hover and interactive UI states

---

## 🛠️ Built With

- HTML5 – Semantic structure
- CSS3 – Styling and responsive layout
- JavaScript (Vanilla JS) – Logic and DOM manipulation
